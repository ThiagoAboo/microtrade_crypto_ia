# OPTIMIZATION PROMPT

Otimize o sistema atual para:

- menor uso de RAM
- menor latência
- menor uso de CPU
- menor uso de disco

Hardware alvo:
- i7-1165G7
- 8GB RAM
- Intel Iris Xe

Prioridades:
1. estabilidade
2. baixa latência
3. eficiência operacional
4. consumo de memória

NÃO faça overengineering.

Evitar:
- abstrações excessivas
- dependências pesadas
- pipelines complexas
- otimizações prematuras

Antes de implementar:
explique o plano de otimização.
aguarde a minha decisão.
